# Study Genie - The Study Resource Recommendation System
AI - powered Recommendation System helps to discover best learning resources incluiding Youtube videos, tutorials, Documentations and Research Papers based on topic and your learning level.

## Features
1. Level based recommendation 
    - School
    - College
    - Research
2. Youtube videos
    - Clickable
    - Channel name
    - Duration
    - Thumbnail
3. Smart Query
    - Auto improve based on level
4. UI
    - Clean layout
    - Responsive

## TechStack
- Language - Python
- Frontend - Streamlit
- Video - yt-dlp
- Framework - Langchain

## Installation
Install dependencies:
```bash
pip install -r requirements.txt
```
Run:
```bash
streamlit run app.py
```
## Workflow
1. Enter topic name.
2. Select learning level.
3. System generates smart search query.
4. Recommends best resource.

## Example
Input:

    Topic: Neural Network 
    Level: College

Output:

     Youtube videos
     Documentations
     REsearch Papers
